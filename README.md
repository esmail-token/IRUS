# IRUS Token

IRUS is a symbolic BEP‑20 token built on the BNB Chain, representing unity and cultural connection between the people of Iran and the United States. This repository contains the official website, branding assets, and technical resources for the IRUS project.

---

## 🔹 Token Information
- Name: IRUS  
- Symbol: IRUS  
- Standard: BEP‑20  
- Network: BNB Chain  
- Total Supply: 419,000,000  

---

## 🔹 Project Vision
IRUS aims to promote unity, transparency, and community‑driven growth. The token symbolizes cultural harmony and collaboration between two nations, focusing on positive global engagement.
